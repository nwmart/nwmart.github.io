Privacy Policy

Last updated: April 20, 2026

This Privacy Policy describes how Martinson Consulting Engineers, LLC ("MCE", "we", "our") handles information accessed through MCE Portal, an internal-use application used solely by authorized MCE personnel.

Scope

MCE Portal is an internal operations tool. It is not offered to the public and does not collect information from external users. This policy exists to satisfy platform requirements for third-party APIs (including Intuit QuickBooks Online) consumed by the application.

Information We Access

When authorized by an MCE user, the MCE Portal accesses data from connected third-party services, which may include:

  • HubSpot: contacts, companies, and sales pipeline records
  • Productive.io: projects, deals, time entries, and invoice metadata
  • Intuit QuickBooks Online: financial reports (Profit & Loss, Balance Sheet), invoice and expense data associated with the authorized QuickBooks company
  • Microsoft 365 / SharePoint: document metadata and contents accessible to the authenticated user

How We Use Information

Information accessed is used solely to render internal dashboards, reports, and reference material for MCE personnel. It is not sold, rented, shared, or transferred to third parties.

Data Storage

Data retrieved from third-party APIs is held in memory or cached locally for the duration required to render requested views. OAuth credentials (tokens and refresh tokens) are stored in protected configuration on devices operated by MCE and are never transmitted to third parties other than the originating service for token refresh.

Data Sharing

We do not share information accessed through this application with any party outside MCE, except:

  • As required by law
  • To the originating third-party API solely for the purpose of refreshing access credentials

Security

Access credentials are stored in files excluded from version control. Application code is maintained in private source repositories. Data in transit uses TLS connections to the originating API providers.

Intuit QuickBooks Online Compliance

MCE Portal integrates with Intuit QuickBooks Online. In accordance with Intuit's Developer Code of Conduct, we:

  • Access QuickBooks data only with explicit user authorization
  • Use the data solely for the purpose disclosed to the user at the time of authorization
  • Allow the user to disconnect the integration at any time through Intuit's or MCE's interfaces
  • Retain no data in persistent storage beyond operational caching

Changes

We may update this policy from time to time. Material changes will be reflected in the "Last updated" date above.
